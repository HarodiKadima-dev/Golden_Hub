# ⚽ Golden Hub

> Vitrine oficial dos maiores vencedores do futebol mundial

## 📌 Sobre o projeto
Plataforma que apresenta de forma organizada todos os vencedores das duas principais distinções individuais do futebol:
- 🥇 **Bola de Ouro** — mostra **nome do jogador, ano da conquista e destaque da sua época**
- 🥾 **Chuteira de Ouro** — mesma estrutura: nome, ano e principais marcas registadas

Informação clara, ordenada e fácil de consultar.

## ✨ Características
✅ Estrutura organizada por categorias  
✅ Dados completos e resumidos de cada conquista  
✅ Navegação simples e intuitiva  
✅ Layout responsivo — funciona bem em telemóvel, tablet e computador

## 🛠️ Tecnologias utilizadas
- ✅ **HTML5** — estrutura de todas as secções e páginas
- ✅ **CSS3** — estilo, disposição, cores e adaptação a qualquer ecrã
- ✅ **JavaScript** — organização, apresentação e funcionamento dinâmico dos dados
